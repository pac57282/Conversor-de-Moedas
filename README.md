## Iniciando o projeto

Para iniciar o projeto, executar o ```main.dart```

Necessário atualizar o valor do câmbio do dia no arquivo ```Controller.dart``` conforme abaixo:

```
  double real = 1;
  double dolar = 4.96;
  double euro = 5.24;
  double bitcoin = 194121.70;
  ```
